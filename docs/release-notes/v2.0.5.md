# Release Notes: v2.0.5 (Senbazuru Update)

**Release Date:** 2025-12-13
**Codename:** Senbazuru

## 🚀 Highlights

**Senbazuru (千羽鶴)** 象徵著耐心與精細的工藝。此版本專注於 UI 的精細打磨，特別解決了長檔名顯示與佈局溢出的問題，確保在處理大量檔案 (Batch Processing) 時的視覺體驗如同摺紙般整潔完美。

## ✨ New Features

- **Smart Filename Truncation (智慧檔名截斷)**
  - 引入了智慧截斷邏輯 `truncateFilename`，當檔名過長時自動保留關鍵的前 10 個字元與後 4 個字元 (例如 `114423057_....pdf`)。
  - 在 `QueueItem` 與 `QueueSidebar` 全面應用此邏輯，確保資訊清晰且版面整潔。
  - 支援 Tooltip 懸停顯示完整檔名，資訊不漏接。

## 🐛 Fixes

- **Layout Overflow Protection**
  - 修復了在多檔案佇列中，Badge 與 Action 按鈕可能被擠壓或溢出容器的問題。
  - 為佇列項目容器添加了 `overflow-hidden` 與 `shrink-0` 約束，確保在各種視窗寬度下的穩定性。

## 🏗 Architecture & Engineering

- **Unit Testing**: 新增了針對截斷邏輯的完整單元測試 (`truncateFilename.test.ts`)，覆蓋了邊界情況與參數調整驗證。
